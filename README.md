# 微信公众号菜单编辑器
 * 微信公众号自定义菜单编辑器，适用于认证订阅号、服务号和测试公众号。使用NwJs+NodeJs+AngularJs开发成桌面版，可配置、查询、编辑、发布公众号菜单。
